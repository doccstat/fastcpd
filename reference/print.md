# Print the call and the change points for a [fastcpd](https://fastcpd.xingchi.li/reference/fastcpd-class.md) object

Print the call and the change points for a
[fastcpd](https://fastcpd.xingchi.li/reference/fastcpd-class.md) object

## Usage

``` r
# S3 method for class 'fastcpd'
print(x, ...)

# S4 method for class 'fastcpd'
print(x, ...)
```

## Arguments

- x:

  A [fastcpd](https://fastcpd.xingchi.li/reference/fastcpd-class.md)
  object.

- ...:

  Ignored.

## Value

Return a (temporarily) invisible copy of the
[fastcpd](https://fastcpd.xingchi.li/reference/fastcpd-class.md) object.
Called primarily for printing the change points in the model.
