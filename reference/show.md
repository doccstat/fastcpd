# Show the available methods for a [fastcpd](https://fastcpd.xingchi.li/reference/fastcpd-class.md) object

Show the available methods for a
[fastcpd](https://fastcpd.xingchi.li/reference/fastcpd-class.md) object

## Usage

``` r
# S3 method for class 'fastcpd'
show(object)

# S4 method for class 'fastcpd'
show(object)
```

## Arguments

- object:

  A [fastcpd](https://fastcpd.xingchi.li/reference/fastcpd-class.md)
  object.

## Value

No return value, called for showing a list of available methods for a
[fastcpd](https://fastcpd.xingchi.li/reference/fastcpd-class.md) object.
