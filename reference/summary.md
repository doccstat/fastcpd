# Show the summary of a [fastcpd](https://fastcpd.xingchi.li/reference/fastcpd-class.md) object

Show the summary of a
[fastcpd](https://fastcpd.xingchi.li/reference/fastcpd-class.md) object

## Usage

``` r
# S3 method for class 'fastcpd'
summary(object, ...)

# S4 method for class 'fastcpd'
summary(object, ...)
```

## Arguments

- object:

  A [fastcpd](https://fastcpd.xingchi.li/reference/fastcpd-class.md)
  object.

- ...:

  Ignored.

## Value

Return a (temporarily) invisible copy of the
[fastcpd](https://fastcpd.xingchi.li/reference/fastcpd-class.md) object.
Called primarily for printing the summary of the model including the
call, the change points, the cost values and the estimated parameters.
